Modulo Formulario Joomla ( Soluciones Vigo )
Un modulo tiene los siguientes campos:
- Nombre
- Telefono
- Departamentos ( Donde puede indicar a que departamentos vas enviar la consulta formulario : ayuda,venta,facturacion)
- Lopd -> solo texto
- captcha -> muestra un pregunta de suma de dos numeros.
- Check de renvio -> Un Check de reenvio de formulario al email que indique. [ Pendiente controlar esto ]

Ademas el formulario permite seleccionar:
- Si se activa (muestra) cada campo o no
- Si es obligatorio o no cada campo.
- Si se activa filtro de palabra ( anti-spam) 

La palabras del filtro tiene que estar separadas por punto y coma (;)
